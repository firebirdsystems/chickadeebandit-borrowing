# Borrowing

A [Chickadee Bandit](http://chickadeebandit.com) app.

Request to borrow items from family members with agreed return conditions, dates, and consequences. Both parties must approve before the agreement is locked — any edit resets it and requires re-approval.

## Features

- Submit borrow requests with item, return date, and agreed consequences
- Two-party approval flow — lender and borrower both confirm
- Full history of active, pending, and past agreements
- Alerts when an agreement is reached or an item is returned

## Install

In your hub, go to **Apps → Install from URL** and paste:

```
https://github.com/firebirdsystems/chickadeebandit-borrowing/releases/latest/download/bundle.json
```
